Enable Windows Embedded 2009 updates

Configures Windows to appear as Windows Embedded POSReady 2009 to Windows Update, enabling access to Windows XP security updates released between 2014 and 2019.

Microsoft officially advises against doing this, as Windows Embedded 2009 updates were not tested against Windows XP. However, Legacy Update users have rarely run into issues installing or using these updates. Visit our Frequently Asked Questions for further information.
