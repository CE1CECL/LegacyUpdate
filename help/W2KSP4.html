Windows 2000 Service Pack 4

This installs Windows 2000 Service Pack 4, and Update Rollup 1 for Windows 2000 Service Pack 4. This brings your operating system up-to-date to April 2005. Further updates can be installed via the Legacy Update website.

Your computer will restart automatically to complete installation.

Description

Update Rollup 1 for Windows 2000 SP4 was released June 28, 2005. This update rollup contains a list of security-related updates produced for Windows 2000 between the release of Windows 2000 SP4 and April 30, 2005.
