Windows XP Service Pack 3

This installs Windows XP Service Pack 3. This brings your operating system up-to-date to April 2008. Further updates can be installed via the Legacy Update website.

If you are running the original release of Windows XP (RTM, or Release to Manufacturing), Service Pack 1a will be installed first.

Your computer will restart automatically to complete installation.

Description

Windows XP Service Pack 3 (SP3) is a free update for Windows XP. SP3 includes all previously released Windows XP updates, including security updates, hotfixes, and select out-of-band releases.
